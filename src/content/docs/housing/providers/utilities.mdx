## Важная информация

:::note
В Испании вы можете свободно выбирать поставщика энергии. При смене поставщика:
- Не требуется менять счетчики или проводку
- Нет перерыва в подаче электричества или газа
- Процесс занимает около 15-20 дней
:::

### Типы тарифов

1. **Регулируемый тариф (PVPC)**
   - Цена меняется каждый час
   - Обычно выгоднее для малого потребления
   - Доступен только для электричества

2. **Свободный рынок**
   - Фиксированная цена на период контракта
   - Может включать дополнительные услуги
   - Доступен как для электричества, так и для газа

### Необходимые документы

Для заключения контракта потребуются:
- DNI/NIE
- CUPS (уникальный код точки подключения)
- Банковские реквизиты
- Документ о праве пользования помещением (контракт аренды или документ о собственности)
